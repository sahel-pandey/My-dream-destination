# My Dream Destination: USA
LIve Link:https://sahel-pandey.github.io/My-Dream-Destination/
A website dedicated to exploring the top travel destinations and attractions within the United States.

## 📋 Overview

This project showcases some of the most beautiful and iconic places to visit in the USA, providing information about top destinations, their attractions, and useful resources for travelers.

## 🏠 Pages

### Home Page (index.html)
- **Description**: Main landing page featuring USA travel information
- **Content**:
  - Overview of the USA with description
  - Featured image of the USA
  - Top 5 places to visit (ordered list)
  - Table of attractions by city
  - Useful external links to Wikipedia and White House

### Contact Page (public/contact.html)
- **Description**: Contact form for visitor inquiries
- **Features**:
  - Contact form with name, email, and message fields
  - Form submission capability
  - Link back to homepage

## 🎯 Top Places Featured

1. **New York City** - Statue of Liberty, Central Park, Times Square
2. **Grand Canyon** - Hiking, Rafting, Scenic Views
3. **Yellowstone National Park** - Geysers, Wildlife, Hiking Trails
4. **San Francisco**
5. **Las Vegas**

## 🚀 Getting Started

1. Open `index.html` in your web browser to view the website
2. Navigate to the contact page to submit inquiries
3. Explore the different sections to learn about USA destinations
